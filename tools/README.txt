export-from-R77.30.ps1 - скрипт PowerShell для экспорта объектов и правил в формате XML из Security Management Server (SMS) Check Point версии R77.30.

По сути это скрипт-обёртка над утилитой cpdb2web официально распространяемой компанией Check Point.

При запуске, скрипт потребует ввести IP-адрес сервера управления, название экспортируемой политики, имя пользователя и пароль.

После отработки скрипта, все экспортированные объекты и правила будут сохранены в созданном каталоге objects-and-rules.

При необходимости, в теле скрипта (строка 27) нужно поменять путь до утилиты C:\Software\cpdb2web\cpdb2web.exe.
